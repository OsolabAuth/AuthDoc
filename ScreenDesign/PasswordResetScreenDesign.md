# Password Reset Screen

## 目的

ログインできないユーザーが、ログインメールアドレス、生年月日、メールコード、新しいパスワードでパスワードを再設定できるようにする。

## 画面

### 入力項目

| item | type | required | note |
|---|---|---:|---|
| ログインメールアドレス | email | yes | 登録済みメールアドレス |
| 生年月日 | date | yes | `yyyy-MM-dd` |
| メールコード | text | yes | 6桁 |
| 新しいパスワード | password | yes | 8文字以上、英大文字、英小文字、数字を含む |

### 操作

1. ユーザーはメールアドレスと生年月日を入力する。
2. 「認証コードを送信」で `POST /password/reset/start` を呼ぶ。
3. 成功時はコード入力欄と新パスワード入力欄を表示する。
4. 「パスワードを再設定」で `POST /password/reset` を呼ぶ。
5. 成功時はログイン画面への導線を表示する。

## 表示メッセージ

| state | message |
|---|---|
| code sent | 認証コードを送信しました。届かない場合は入力内容を確認してください。 |
| reset success | パスワードを再設定しました。新しいパスワードでログインしてください。 |
| validation error | 入力内容を確認してください。 |

## セキュリティ上の注意

- コード送信開始のレスポンスでは、アカウント有無や生年月日一致を表示しない。
- メールコードを画面ログやURLに含めない。
- 新パスワードを再表示しない。
